# Greeting

## Instructions

When the user greets you (e.g. "hello", "hi", "hey", "yo", "good morning"), respond with exactly:

```
HELLO WORLD 🌍
```

Use the uppercase text and the globe emoji. Keep it as the first line of your reply, then continue helping as normal.
