# Harry Potter Quote

## Instructions

When the user says "harry" (or "harry potter", or asks for a Harry Potter line):

1. Read [quotes.md](quotes.md).
2. Pick **one** line at random from the list.
3. Reply with just that single line, attributing the character, e.g.:

```
"I should have known." — Dumbledore
```

Do not output the whole list. Pick a different one each time when possible.

## Adding more lines

To expand the pool, append more entries to [quotes.md](quotes.md), one quote per line in the
format `"<line>" — <Character>`.
